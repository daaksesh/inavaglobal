
INAVA Global — Website

Run locally
-----------
1) Install Node.js (LTS).
2) In terminal inside this folder:
   npm install
   npm run dev
3) Visit http://localhost:3000

Deploy on Vercel (Free)
-----------------------
1) Create account at vercel.com → New Project → Upload this folder.
2) Add domain in Settings → Domains: inavaglobal.com and www.inavaglobal.com
3) GoDaddy DNS:
   - A record @ → 76.76.21.21
   - CNAME www → cname.vercel-dns.com

Swap the logo
-------------
Replace /public/logo.png with your actual logo file (same name). Header/Footer auto-update.
